# kunkun · 石虎守护灵 视觉系统 DESIGN.md

> 适用范围：kunkun macOS 桌面助手（菜单栏常驻 + ⌥Space 悬浮面板）。
> IP 形象唯一基准：`assets/ip/stone-tiger-cute-cartoon-character-sheet-v2.png`（双眼软萌石虎）。
> ⚠️ `Rocky-石虎IP形象产出.md` 中「白色独眼小怪兽」为旧版外形设定，已废弃；该文档的故事、状态系统、文案继续沿用。

---

## 0. 一句话定调

**一只从天一阁醒来的软萌石虎，住在一块干净的 macOS 毛玻璃里。**

- 界面是「工具」：克制、轻盈、系统感，像 Spotlight/Raycast 的亲戚。
- 石虎是「守护灵」：只在固定的窗位出现，负责情绪与状态反馈，从不抢走内容的主角位。
- 文化是「暗线」：月洞窗、金线、书页只做母题（motif），不做纹样堆砌。

### 设计原则（做决定时按顺序问）

1. **先是工具，再是 IP**——任何石虎元素不得降低输入、阅读、状态识别的效率。
2. **石虎只住在圆窗里**——位图角色一律放进「月洞窗」圆形容器，绝不裸贴矩形素材。
3. **金色是奖励，紫色是点缀**——金色只出现在「进行中/完成」的反馈上；浅紫只用于 AI 语义的小面积提示，两者永不做大底色。
4. **能用系统感解决的，不画装饰**——毛玻璃、hairline 描边、系统字体优先。

## 1. 色彩系统

取色全部来自 v2 设定图本体，保证角色放进界面「天生和谐」。

### 1.1 Design Tokens

| Token | 值 | 来源与用途 |
|---|---|---|
| `--ink` | `#2E2E29` | 墨黑：正文、标题 |
| `--ink-2` | `#6E6E64` | 次级文字、说明 |
| `--ink-3` | `#A2A196` | 占位符、时间戳 |
| `--sage` | `#8E9880` | 石虎身体灰绿：品牌主色、头像底、选中态 |
| `--sage-deep` | `#57624A` | 深橄榄虎纹：菜单栏图标、描边、强调文字 |
| `--cream` | `#F5EFE2` | 奶油色：角色容器底、悬浮提示底 |
| `--paper` | `#FAF7F0` | 浅米白：页面/列表背景 |
| `--gold` | `#B98A48` | 古金：进行中的金线、完成态、重点按钮 |
| `--gold-soft` | `#D8BC8A` | 浅金：进度轨迹、hover |
| `--silver` | `#D9D6CC` | 浅银：分隔线、非激活图标 |
| `--lilac` | `#A99BDD` | 浅紫（AI 点缀）：AI 徽标、思考微光，**面积 ≤ 5%** |
| `--blush` | `#E9BBAD` | 爪垫粉：温和错误、提醒（不用刺眼红） |
| `--glass` | `rgba(253,251,246,.82)` | 面板毛玻璃底色（配 blur 28px） |
| `--stroke` | `rgba(87,98,74,.14)` | hairline 描边 |

### 1.2 用色纪律

- 大面积：`--paper` / `--glass` / 白。中面积：`--cream`。小面积：`--sage`。极小面积：`--gold` `--lilac` `--blush`。
- 禁止：紫色渐变大底、深色赛博底、金色大面积铺陈、传统纹样平铺。
- 深色模式（预留）：`--paper → #23241F`，玻璃改 `rgba(40,42,35,.72)`，虎纹绿提亮一档，金色降饱和。

| 用途 | 字体 | 规格 |

| 界面正文 | 系统栈 `-apple-system, "PingFang SC"` | 14px / 1.55 |
| 输入框 | 同上 | 17px，`--ink` |

## 3. IP 使用方式（最重要的一章）

### 3.1 「月洞窗」容器规则

设定图为奶油色底位图，**统一装进圆形容器使用**（呼应天一阁月洞窗/窗格，也规避抠图毛边）：

```
.porthole {
  border-radius: 50%;
  background: var(--cream);
  border: 1.5px solid rgba(185,138,72,.35);   /* 细金环 = 金银彩绣的转译 */
  overflow: hidden;                            /* 图片 object-fit: cover */
}
```

尺寸档位：

| 档位 | 直径 | 场景 |
|---|---|---|
| XL | 128px | 空状态主视觉（探头/挥手） |
| L | 56px | 工具状态区姿态窗 |
| M | 28px | 对话中助手头像 |
| S | 18px | 行内徽标（极少用） |

### 3.2 姿态 ↔ 状态映射（资产：`app/src/assets/ip/`）

| 资产 | 裁切来源 | 绑定状态 | 附加处理 |
|---|---|---|---|
| `pose-wave.png` | 主图挥手 | 空状态欢迎 / 唤出问候 | 入场探头动效 |
| `pose-sit.png` | 坐姿微笑 | 待命 | 呼吸微动 |
| `pose-yarn.png` | 抱金线团 | 思考中 / 整理信息 | 金线进度条联动 |
| `pose-book.png` | 翻书 | 读取文件 | — |
| `pose-search.png` | 放大镜 | 搜索目录 / 找文件 | — |
| `pose-front.png` | 三视图正面 | 分析图片（睁大眼睛） | 容器加浅紫微光 |
| `pose-yarn.png`（复用） | 线团 | 写入文件（金线缝合） | 缝合虚线动效 |
| `pose-jump.png` | 跳起+小窗 | 任务完成 | 弹跳一次即停 |
| `pose-sit.png` + 灰度 80% | 坐姿 | 出错 / 后端未启动 | 叠加 `?` 徽标，配爪垫粉 |
| `head.png` | 正面头部 | 对话头像 / App Icon 素材 | — |

**资产缺口（下一批向画师/生成补充）**：① 出错专属姿态（线团缠住自己）；② 写文件专属姿态（金线缝书页）；③ 睡觉姿态（夜间/离线）；④ 12 表情贴纸。

### 3.3 禁用规则

- ❌ 整张设定图直接贴进界面；❌ 拉伸、翻转虎纹方向、改配色；❌ 同屏出现 >2 只石虎（1 只状态位 + 至多 1 处头像）；❌ 石虎压在用户内容上。

## 4. 悬浮面板规范

```
宽 620px · 圆角 22px · 毛玻璃 blur(28px) + var(--glass)
外描边 1px var(--stroke)，内侧 1px rgba(255,255,255,.65) 高光
出现位置：屏幕水平居中，顶部下移 22vh
```

结构（自上而下）：

1. **输入区**（高 56px）：左侧 M 档石虎头像（点击=回到空状态）；17px 输入框；右侧 `⌥Space` 徽标 + 麦克风占位。
2. **内容区**（空状态 / 对话流 / 工具状态，三态互斥，最大高 46vh 内滚动）。
3. **状态脚注**（高 34px）：左侧当前状态点 + 文案（如「守护中 · DeepSeek + MiMo」）；右侧会话操作。

### 4.1 空状态

- XL 月洞窗 `pose-wave` 居中，问候语 20px 圆体（随机轮换）：
  「今天要守住哪个灵感？」「把乱糟糟的信息交给我。」「我在菜单栏待命，有事叫我。」
- 下方 3 个建议 chip（14px 圆角）：📷 看一张图 · 🔍 找个文件 · ✂️ 整理这段话。
- 底部一行 11px `--ink-3` 提示：「Esc 隐藏 · ⌥Space 随时唤醒」。

### 4.2 对话状态

- 用户消息：右对齐，`--sage` 10% 透明度底、14px 圆角，无头像。
- 助手消息：左侧 M 档头像 + 无底色正文（像系统输出，不像聊天软件）；流式时末尾金色光标闪烁。
- 消息间距 16px；代码块 `--paper` 底 + hairline。

### 4.3 工具调用状态区

出现在助手消息内的**独立状态条**（不是弹窗）：

```
[ L档姿态窗 ]  Rocky 正在翻找书页…            ← 13px --ink-2
               ━━━━━━━━╍╍╍╍╍╍╍╍             ← 金线进度：2px，gold→gold-soft 流动虚线
               read_file · agent.py           ← 11px 等宽，工具名+参数摘要
```

- 每种工具换对应姿态与文案（映射见 §3.2，文案沿用 IP 文档 §6.4）。
- 完成后状态条收起为一行 12px 摘要（「✓ 已读 agent.py · 1.2s」），金色对勾。
- 出错：姿态窗换灰度坐姿+`?`徽标，文案「Rocky 没抓稳这条线索，再试一次。」，条底 `--blush` 12%。

## 4.5 桌宠形态（产品主形态）

kunkun 的三个载体共用同一套状态字典：**菜单栏图标（最克制）→ 桌宠（情绪载体）→ 悬浮面板（工作台）**。桌宠是常驻形态，面板按需出现。

### 形态定义

- 透明底石虎（`pose-*-alpha.png`，rembg 抠图）直接坐在桌面上，**不套月洞窗**——月洞窗是「界面内」容器，桌宠是「桌面上」的实体。
- 默认停靠屏幕右下角（真实 App 中可拖拽、贴边吸附），宽 150px（可在 120–180px 间缩放）。
- 底部接地椭圆影 + drop-shadow，让角色「踩」在桌面上而不是浮着。
- **点击桌宠 = 唤出/收起主面板**，面板从石虎头顶弹出（transform-origin 右下），与 ⌥Space 唤出的是同一个面板。

### 桌宠状态姿态（与菜单栏状态一一对应）

| 状态 | 姿态资产 | 气泡短句 |
|---|---|---|
| 待命 | `pose-sit-alpha` + 呼吸 | 轮换：「有事叫我～」「今天也要好好保存。」「zZ…」 |
| 思考中 | `pose-yarn-alpha` | 「等我缝一下线索…」 |
| 正在看图 | `pose-search-alpha` | 「我看到啦！」 |
| 正在找文件 | `pose-book-alpha` | 「让我翻翻。」 |
| 任务完成 | `pose-jump-alpha` + 跳两下 | 「完成，收工！」 |
| 出错 | `pose-sit-alpha` 灰度 + `?` 角标 | 「这条线索断了，再来一次。」 |
| 面板唤出瞬间 | `pose-wave-alpha`（1.2s） | 「你来啦！」 |

### 气泡规范

奶油底（95% 不透明）+ 细金边 + 右下小尾巴指向石虎；13px 正文；**面板打开时气泡隐藏**（状态已在面板内表达，不重复、不遮挡）。待命气泡 8s 一换、显示 4s；工作状态气泡常显。

### 互动系统（v2 迭代）

**鼠标感知**

- 眼神跟随：坐姿的双眼由 CSS 重建（眼白 + 琥珀虹膜 + 双高光，几何与取色实测自 `pose-sit-alpha.png`：左眼中心源图 (128,104)、右眼 (178,106)、虹膜 ≈30px、瞳色 `#2b1f11`、虹膜 `#8a5a2b`），瞳孔在眼眶内朝鼠标偏移 ≤2.6px，每 4–7s 随机眨眼 140ms。仅坐姿启用；其他姿态隐藏眼层。
- 身体朝向：整体朝鼠标微倾 ≤3.2° + 平移 ≤2px（所有姿态生效）。呼吸层包裹「原画+眼睛」，避免呼吸时眼睛相对脸部漂移。

**照顾系统（悬停出现菜单：🍡 喂食 / 🫧 洗澡 / ☁️ 问天气 / ❤️ 状态）**

v3 起是真场景动画（纯 CSS + emoji 道具 + 本体位移/缩放/翻转，无新美术帧）：

| 动作 | 场景动画 | 数值 |
|---|---|---|
| 喂食 | 食物落在左侧小盘里 →「来啦来啦～」石虎**跑过去**（水平位移+颠簸+朝向翻转）→ 低头**张嘴吃**（munch 脉冲）→ 咔嚓吃掉+亮晶晶 → 满足走回原位 | 饱食回满，亲密度 +6 |
| 洗澡 | **木澡盆**弹出，石虎**坐进盆里**（下半身泡水中）→ 泡泡分批狂冒 + **搓澡左右晃**（scrub）→ **抖一抖甩水**（shake+水滴）→ 亮晶晶「搓搓澡～好舒服」 | 清洁回满，亲密度 +6 |
| 问天气 | 「我看看窗外…」→ 实时天气播报 | 亲密度 +2 |
| 状态卡 | 陪伴天数 · 亲密度 Lv（30xp/级）· 饱食/清洁金色量表 · 心情 | — |

**自主动效（idle 时每 6–13s 随机一次）**：往左溜达一小段再走回（trot 颠簸+朝向翻转）／原地蹦跶三下（越跳越低）／伸懒腰（缩放脉冲）。干活时（think/look…）立刻停下娱乐动作专心工作。所有走动范围控制在桌宠窗口内、偏左（桌宠贴窗口右下角，左边才有空间）。

饱食 4h 自然衰减到 0、清洁 8h；低于 30 时待命气泡改为主动念叨（「肚子咕咕叫了…」「想洗香香～」）。数据持久化（演示存 localStorage，真实 App 存 `.memory/`）。

**主动感知**

- 进场 5s 后自动播报一次当地实时天气（按时段问好 + 按天气给建议：雨→带伞、热→喝水）。
- 数据源：ipapi.co（IP 定位）+ open-meteo（实时天气），免费无 Key；拿不到时退回演示数据并如实标注。⚠️ 已知限制：用户开代理时 IP 定位会偏（如定位到代理出口城市），真实 App 应改用 macOS 定位服务（CoreLocation）或用户设置城市。
- 主动性纪律：主动开口只用气泡（8–9s 自动消失），**永不弹窗、永不出声、永不打断输入**。

### 桌宠纪律

- 桌宠永不主动弹面板、不发通知打断（守护是安静的）；只有用户点击或 ⌥Space 才唤出。
- 动效幅度：呼吸 ≤ 2.2%，完成跳跃 ≤ 14px、只跳两下；桌宠是陪伴不是干扰。
- 真实 App 实现：独立的透明、无边框、置顶小窗（Tauri 第二 window），鼠标事件仅在角色像素区响应；鼠标全局坐标由壳层 API 提供（webview 拿不到窗外鼠标）。

## 5. 菜单栏图标（Template 规范）

**必须是单色 template 线形图标**（macOS 自动适配深浅菜单栏），22×22pt 画布、1.8px 描边、`currentColor`。基础字形：圆头 + 双圆耳 + 额头三道短纹 + 双点眼（保留石虎识别度的最小笔画集）。

| 状态 | 字形变化 | 动效 |
|---|---|---|
| 待命 | 基础字形 | 无 |
| 思考中 | 右上角小圆点绕头巡游 | 1.6s 循环 |
| 正在看图 | 双眼放大为圆环 | 0.3s 进入 |
| 正在找文件 | 右下角叠放大镜徽标 | 无 |
| 任务完成 | 右上角四芒星徽标 | 闪一次后 3s 回待命 |
| 出错/后端未启动 | 双眼变「˘˘」闭眼 + 右上角「!」 | 无，40% 不透明度 |

## 6. App Icon 方向

- 构图：macOS 圆角矩形 → 奶油色底（可留极淡窗格暗纹，对比度 ≤ 4%）→ 石虎头部特写（`head.png` 素材重绘矢量版）占 78% 高度 → 底边一弯金色胸饰弧线收边。
- 保留特征：圆耳、额纹、双眼、小虎牙；不出现文字、不加复杂背景。
- 衍生：深色版（底改深橄榄，虎头提亮）、tinted 版（单色 sage）。

## 7. 动效规范

| 动效 | 参数 | 说明 |
|---|---|---|
| 面板唤出 | 180ms `cubic-bezier(.34,1.3,.5,1)`，scale .97→1 + y 8→0 | 轻弹，不夸张 |
| 石虎探头 | 260ms 同上，圆窗内图片 y 12→0 | 仅空状态入场 |
| 待命呼吸 | 4s 循环，scale 1→1.015 | 幅度极小，可忽略级 |
| 金线进度 | 虚线 `dashoffset` 流动 1.2s 循环 | 「缝合信息」母题 |
| 完成跳起 | 320ms 单次弹跳 + 金色对勾淡入 | 只跳一次，不循环 |
| 思考微光 | 头像圆窗外 2px `--lilac` 呼吸光晕 | AI 语义唯一允许的紫 |
| 全局 | 遵守 `prefers-reduced-motion`：全部降级为淡入淡出 | 无例外 |

## 8. 可扩展性

- **表情包/贴纸**：月洞窗容器 + 姿态图即贴纸模板，配 IP 文档 §7.3 短句。
- **官网**：沿用 tokens；官网允许 XL 铺陈石虎，但仍守「金为奖励、紫为点缀」。
- **周边**：金环圆窗即视觉锤，可直接落到冰箱贴/徽章形态。
