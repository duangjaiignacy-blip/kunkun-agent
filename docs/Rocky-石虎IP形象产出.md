# Rocky 石虎 IP 形象产出

> 项目场景：kunkun macOS 桌面智能助手  
> IP 核心：以 Rocky 的独眼小怪兽形象为主视觉，将“天一阁石虎”的守护精神转译为年轻化、治愈型、系统级桌面助手角色。

---

## 1. IP 定位

### 1.1 一句话定位

**Rocky 是从天一阁石虎文化中苏醒的新一代桌面守护灵，陪用户在电脑里整理信息、观察世界、守住灵感。**

### 1.2 IP 类型

- **文化来源**：天一阁石虎、藏书楼守护文化、金银彩绣工艺
- **产品场景**：macOS 桌面智能助手、菜单栏常驻、Spotlight 式悬浮面板
- **情绪价值**：陪伴、治愈、守护、机灵、轻松
- **传播方向**：AI 助手人格化、桌面陪伴角色、数字文创 IP

### 1.3 设计关键词

**守护感 / 机灵感 / 软萌感 / 文化转译 / 桌面陪伴 / 灵感捕捉 / 情绪安定**

---

## 2. IP 名称与身份

### 2.1 中文名建议

**洛奇 Rocky**

辅助命名可根据不同场景使用：

- **醒石 Rocky**：偏文化叙事，适合毕业设计、作品集、品牌故事
- **石虎小怪 Rocky**：偏可爱传播，适合表情包、周边、社交平台
- **困困守护灵 Rocky**：偏产品陪伴，适合 macOS App 内部角色

### 2.2 角色身份

Rocky 不是普通小怪兽，而是由天一阁石虎守护精神转译而来的新生守护灵。

它原本沉睡在天一阁石阶、飞檐与藏书楼的光影之间，长期守护书籍、文字和人的记忆。后来，一束穿过针孔的光照进旧墙，一缕金银彩绣的线缠绕在它身上，让它从石虎形态中苏醒，变成更适合当代年轻人亲近的圆润小怪兽。

来到电脑世界后，Rocky 成为 kunkun 桌面助手的可视化人格。它会观察屏幕、理解图片、整理文件、记住任务，也会在用户疲惫时用轻松的方式提醒、陪伴和鼓励。

## 3. IP 故事

### 3.1 完整故事版

1561 年，天一阁的石虎守在庭院、书楼和碑林之间。它们不说话，却一直看着人们读书、抄写、收藏、离开，又回来。

几百年过去，书页泛黄，墙影斑驳，石虎慢慢沉睡。它梦见飞檐上的光，梦见窗格间的风，也梦见一根根金银丝线穿过时间，把破碎的记忆重新缝合。

直到某天，一束像针孔一样细小的光照在它的眼睛上。金银彩绣的线从旧时光里亮起，缠绕住石虎的角、爪和身体。沉睡的守护灵醒了。

它不再是沉重的石雕，而变成了圆圆软软、只有一只大眼睛的小怪兽 Rocky。

Rocky 的独眼像一枚小小的镜头，能看见屏幕里的图片、文件和文字；它的紫色眼睛保存着天一阁夜色里的光；它的小角和小爪保留着石虎守护者的痕迹；它柔软的身体，则让古老文化变得更容易被亲近。

后来，Rocky 住进了 Mac 的菜单栏。用户按下快捷键，它就从屏幕中央探出头，帮你找文件、看图片、整理想法、拆解任务。它仍然在守护，只是从守护藏书楼，变成守护你的数字记忆和每天的灵感。

### 3.2 短故事版

Rocky 是从天一阁石虎中苏醒的桌面守护灵。  
它被一束针孔之光和一缕金银彩绣唤醒，化成圆润可爱的小怪兽，住进 Mac 菜单栏。  
它用独眼观察世界，用小爪整理信息，用快乐能量陪你完成每天的工作。

### 3.3 Slogan

首选：

**一眼看见灵感，一爪守住日常。**

备选：

- **从天一阁醒来，住进你的 Mac。**
- **石虎醒了，灵感有了守护者。**
- **按下快捷键，Rocky 来守护你的灵感。**
- **针孔映虎，灵感成形。**
- **小怪兽在菜单栏，替你守住每一个想法。**

## 5. 视觉转译

### 5.1 现有形象优势

Rocky 当前形象具备非常强的亲和力：

- 白色圆润身体降低距离感，适合桌面陪伴类产品
- 紫色独眼有清晰记忆点，适合做 App 图标和加载动画
- 小角、小牙、小爪保留怪兽感，也能连接“石虎”的守护属性
- 3D 软萌风格适合延展为毛绒、公仔、贴纸和数字动画

### 5.2 与石虎文化的对应关系

| Rocky 视觉元素 | 石虎文化转译 |
|---|---|
| 小角 | 石虎耳部和守护兽轮廓的年轻化处理 |
| 小牙 | 虎的力量感被弱化为可爱符号 |
| 小爪 | 守护、巡逻、整理书页的动作记忆 |
| 独眼 | 针孔成像、镜头、观察、AI 视觉能力 |
| 紫色眼睛 | 天一阁夜色、数字光源、智慧感 |
| 白色身体 | 石质感的柔软化表达，从“石雕”变成“陪伴物” |
| 圆润体态 | 把传统文化从庄重叙事转译为年轻人的治愈审美 |

### 5.3 建议新增专属符号

为了让 Rocky 更像“天一阁石虎 IP”，建议在现有形象上增加 3 个轻量符号：

1. **金银线环**
   - 位置：手腕、角根部、腰侧或尾部
   - 含义：金银彩绣唤醒 Rocky，也象征信息被缝合整理

2. **石纹小印记**
   - 位置：背后、脚底或耳角
   - 含义：保留石虎来源，不破坏整体软萌感

3. **书页/窗格形眼高光**
   - 位置：独眼高光里
   - 含义：让“看见”和“藏书楼”发生视觉关联

## 6. 产品内角色系统

Rocky 在 macOS App 里不只是品牌图，它应该承担“状态反馈”的功能。

### 6.1 菜单栏图标

建议使用 Rocky 的独眼或小角轮廓作为菜单栏图标。

状态建议：

| 状态 | 图标表现 |
|---|---|
| 待命 | 单只紫色眼睛静止 |
| 思考中 | 眼睛轻微转动或出现光点 |
| 工具调用中 | 小爪/线团动效 |
| 任务完成 | 眼睛高光变亮 |
| 出错 | 眼睛眯起，出现小问号 |
| 离线/后端未启动 | 眼睛闭合或变灰 |

### 6.2 悬浮面板

悬浮面板出现时，Rocky 可以从输入框边缘探头，形成“被召唤”的感觉。

建议交互：

- 唤出面板：Rocky 探出头，看向输入框
- 用户输入：Rocky 眼睛跟随光标
- AI 思考：Rocky 抱着线团整理线索
- 工具调用：Rocky 小爪翻书、找文件、看图片
- 回答完成：Rocky 轻轻挥手或点头
- 用户中断：Rocky 停下动作，回到待命

### 6.3 空状态文案

用于面板首次打开、无历史记录、无任务时。

- “今天要守住哪个灵感？”
- “按下问题，我来看看。”
- “把乱糟糟的信息交给我。”
- “我在菜单栏待命，有事叫我。”
- “想法刚冒头的时候，最值得被抓住。”

### 6.4 工具调用文案

保持 Rocky 的人格，但不影响专业感。

| 场景 | 文案 |
|---|---|
| 读取文件 | Rocky 正在翻找书页... |
| 搜索项目 | Rocky 正在巡逻目录... |
| 分析图片 | Rocky 睁大眼睛看图中... |
| 执行命令 | Rocky 正在动手处理... |
| 写入文件 | Rocky 正在把灵感缝进文件... |
| 任务完成 | Rocky 已经守好了。 |
| 出错 | Rocky 没抓稳这条线索，再试一次。 |

## 7. 表情与动作产出

### 7.1 基础表情

建议先产出 12 个基础表情，覆盖产品和传播两种场景。

1. 开心：默认欢迎状态
2. 认真：正在思考
3. 惊讶：发现问题或新内容
4. 困惑：遇到错误
5. 委屈：请求失败
6. 得意：任务完成
7. 困困：等待太久或夜间使用
8. 加油：鼓励用户继续
9. 嘘声：专注模式
10. 守护：抱住书本或文件
11. 偷看：从窗口边缘探头
12. 发光：灵感被捕捉

### 7.2 动作姿态

建议先产出 10 个动作。

1. Stand：站立待命
2. Wave：挥手召唤
3. Look：睁眼观察
4. Think：抱线团思考
5. Search：拿放大镜/从书堆里找文件
6. Stitch：用金银线缝合信息
7. Guard：抱着书本守护
8. Jump：完成任务后跳起
9. Sleep：菜单栏待机/夜间模式
10. Error：线团缠住自己

### 7.3 表情包短句

- 交给我守住。

## 11. 对外介绍文案

### 11.1 产品内介绍

Rocky 是 kunkun 的桌面守护灵。  
它从天一阁石虎文化中苏醒，住进你的 Mac 菜单栏。  
当你按下快捷键，它会帮你看图、找文件、整理信息、守住灵感。

### 11.2 品牌宣传文案

从前，石虎守着藏书楼。  
现在，Rocky 守着你的数字工作台。  

它用一只眼睛观察屏幕，用一双小爪整理线索，用一缕金线把散落的灵感缝起来。

### 11.3 作品集说明文案

本项目以天一阁石虎文化为原型，将传统守护兽形象转译为面向 macOS 桌面智能助手的年轻化 IP 角色 Rocky。设计通过独眼、小角、小爪、金银线与石纹等视觉元素，建立“观察、守护、整理、陪伴”的角色语义，使 IP 不仅具备文化来源，也能在产品界面中承担状态反馈、情绪陪伴和品牌识别功能。
